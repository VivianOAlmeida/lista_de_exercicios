# Lista de Exercícios de Programação

O Lista de Exercícios de Programação é um repositório criado com o objetivo de praticar lógica de programação e consolidar conceitos fundamentais da programação por meio da resolução de diversos exercícios.

Este projeto funciona como um registro de aprendizado, reunindo atividades que ajudam a desenvolver raciocínio lógico, organização de código e compreensão de estruturas básicas utilizadas no desenvolvimento de software.

Mais do que apenas resolver exercícios, o repositório representa um processo de evolução nos estudos de programação, permitindo acompanhar o progresso ao longo do tempo.

## Funcionalidades

Este repositório contém uma lista de exercícios desenvolvidos com HTML e CSS, com o objetivo de praticar a criação de páginas web, estilização e estruturação de layouts.

### Estrutura Básica

- Criação de uma página HTML com estrutura padrão (DOCTYPE, html, head e body).
- Inserção de título (h1) e parágrafo (p).

### Estilização com CSS

- Aplicação de CSS externo para estilizar elementos da página.
- Alteração de cor e fonte de títulos utilizando Google Fonts (Poppins).
- Listas

Implementação de:

- Lista ordenada com filmes ou séries favoritas.
- Lista não ordenada com hobbies.

### Tabelas

- Criação de uma tabela estruturada para organização de dados.

Uso de colunas como:

- ID do cliente
- Nome e sobrenome
- Email e telefone
- Data de nascimento
- Cidade e estado
- Nível VIP

**Observações**

Inclusão de dados fictícios para demonstração.

### Formulários

Criação de formulários HTML com campos:

- Nome
- E-mail
- Senha
- Botão de envio de dados.
- Imagens
- Exibição de imagem centralizada.
- Adição de título e texto explicativo.
- Estilização com borda de 5px amarela.
- Botões Estilizados

Botões com:

- Fundo verde
- Texto branco
- Cantos arredondados.

### Estrutura de Layout

Criação de layout utilizando divs:

- Topo (azul)
- Conteúdo (cinza)
- Rodapé (preto)
- Bordas Personalizadas

Aplicação de diferentes estilos de borda em uma única div:

- Superior sólida vermelha
- Direita tracejada azul
- Inferior pontilhada roxa
- Esquerda dupla verde

### Fundo Personalizado

- Implementação de gradiente de cores utilizando CSS.
- Fundo ocupando 100% da altura da tela (100vh).

### Animações e Interações

- Botão com mudança de cor ao passar o mouse.
- Efeito de crescimento ao passar o mouse.
- Mudança de cor ao clicar.

### Formulário Estilizado

Formulário com:

- Bordas arredondadas
- Sombra leve
- Fundo cinza claro.

### Layout Responsivo

- Layout de três colunas com Flexbox.
- Cada coluna ocupando 33% da largura da tela.

### Efeitos Visuais

- Aplicação de sombra em texto usando CSS.
- Transições suaves em elementos ao interagir com o mouse.

### Modal (Pop-up)

- Implementação de janela modal centralizada.

- Fundo semitransparente cobrindo a tela.

- Botões de abrir e fechar.

### Transições

- Animação suave de aumento de tamanho em uma div ao passar o mouse.

### Rodapé Fixo

- Rodapé fixado no final da página utilizando CSS.

## Tecnologias

As atividades foram desenvolvidas utilizando:

- HTML5
- CSS3
- Git
- GitHub

## Objetivo Educacional

A prática com listas de exercícios é uma das formas mais eficientes de aprender programação, pois permite:

- Desenvolver pensamento lógico
- Aprender a resolver problemas passo a passo
- Compreender melhor estruturas fundamentais da linguagem
- Treinar organização e escrita de código

Este repositório funciona como um repositório de estudos, onde novos exercícios podem ser adicionados conforme o avanço no aprendizado.
